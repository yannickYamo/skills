# ABX Strategy Framework

A comprehensive Account-Based Everything (ABX) strategy framework for B2B companies with complex sales cycles, high-value deals, and limited addressable markets.

## Install

```bash
npx skills add yannickyamo/skills/abx-strategy
```

Or manually copy `SKILL.md` and the `templates/` folder into your `.claude/skills/abx-strategy/` directory.

## When to Use

- Building GTM strategy for companies with <500 addressable accounts
- Deal sizes exceed $100K+ with 6+ month sales cycles
- Buying committees involve 5+ stakeholders
- Products require technical translation for business buyers
- Marketing must directly influence pipeline, not just awareness

## Frameworks Included

| Framework | Purpose |
|-----------|---------|
| **Marketing as Product** | Treat campaigns as hypotheses with success criteria and kill conditions |
| **Segment-Specific vs. Synergy** | What must differ per segment vs. what's reusable |
| **ABX Account Motions** | DETECT → DECODE → ENGAGE → ACCELERATE → EXPAND |
| **PURE Problem Scoring** | Painful, Urgent, Recognized, Expensive — tier your ICPs |
| **Message Resonance System** | Ensure messages actually land |
| **Capability-to-Benefit Translation** | Technical features → business value |
| **Constant Discovery** | Four questions framework for every deal |
| **Learning Loop Architecture** | Weekly → Monthly → Quarterly feedback cycles |
| **Bet Board Methodology** | Structure every initiative as a testable hypothesis |
| **Budget Allocation by Stage** | Investment mix across market maturity stages |

## Quick Start

Once installed, ask Claude:

```
"Apply the PURE framework to score [target account]"
"Create a capability-to-benefit translation matrix for [product]"
"Design 5 testable messaging bets for [segment]"
"Build a customer journey map for [persona] buying [product]"
"Apply segment-specific vs. synergy analysis to [segments A] and [B]"
```
